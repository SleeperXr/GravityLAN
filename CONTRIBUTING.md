# Contributing to GravityLAN

Thank you for your interest in contributing! GravityLAN is a homelab-first project maintained with a focus on simplicity and utility.

## How to Contribute

1. **Bug Reports**: Open an issue with a clear description and steps to reproduce.
2. **Feature Requests**: Describe the problem you're solving and why it fits the GravityLAN soul.
3. **Pull Requests**:
    - Follow the principles in `SOUL.md`.
    - Keep changes focused and documented.
    - Ensure your code follows the existing style (PEP 8 for Python, Prettier/ESLint for JS).

## Development Setup

See the **Development** section in `README.md` for local setup instructions.

## Security

If you discover a security vulnerability, please refer to `SECURITY.md`.

## Code of Conduct

Be respectful, helpful, and honest. We're all here to build better homelabs.
